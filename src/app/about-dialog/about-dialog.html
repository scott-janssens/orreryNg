About Orrery

What is an Orrery?

An orrery is a mechanical model of the solar system that illustrates the relative positions and motions of the planets and moons. This digital orrery provides an interactive 3D visualization of our solar system using accurate astronomical calculations.

Learn more about orreries on Wikipedia

Technologies

  • Angular 21 for the web application framework
  • Angular Material for UI components
  • AA-JS for astronomical calculations
  • Three.js for 3D rendering
  • Claude Sonnet 4.5 for AI development assistance
